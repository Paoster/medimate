# MediMate Backend Platform

This repository contains the backend and platform services for the MediMate Insurance Platform.

## Features
- AI Document Verification
- Medical Record Management
- Insurance Claim Processing
- Blockchain Integration

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables mapping to `.env` based on `.env.example` if available.

3. Start the development server:
   ```bash
   npm run dev
   ```
